요소를 선택하는 방법

  • 선택자로 선택하는 방법
    1. 요소객체.querySelector("선택자")
      • 하나의 요소를 선택하는 방법
      • 가장 위쪽 아이만 선택됨
    2. 요소객체.querySelectorAll("선택자")
      • 여러 요소를 선택하여 배열 방식으로 쓰는 방법
  • 태그로 선택하는 방법
    1. 태그로 선택 - document.getElementsByTagName("태그명")
      • 배열형태로 선택된다
  • 속성으로 선택하는 방법
    1. document.getElementById("id명")
      • 하나의 객체로 선택된다
    2. documnet.getElementsByClassName("클래스명")
      • 배열형태로 선택된다
  • 요소간의 관계로 선택하는 방법
    1. 노드(텍스트노드, 엘리먼트노드)로 접근한다
    2. 부모
      • 요소객체.parentNode
    3. 형제
      • 다음형제 - 요소객체.nextSibling
      • 이전형제 - 요소객체.previousSibling
    4. 자식
      • 요소객체.childNodes
      • 마지막자식 - 요소객체.lastChild
      • 자식들 중 요소만 - 요소객체.children
      • 요소 중 마지막 자식 - 요소객체.lastElementChild
