자바스크립트 이벤트

이벤트를 적용시키는 방법
  • 인라인방식
    1. 태그 안쪽에 이벤트 속성을 이용하여 추가하는 방법
    2. 예) onclick = "실행할 함수", onfocus = "실행할 함수"
  • 객체의 이벤트 속성에 함수를 추가하는 방법
    1. 요소.on이벤트명 = 실행할함수 정의
    2. 예) element.onclick = function(){ alert(); }
  • addEvent 함수를 이용하여 추가하는 방법
    1. 요소.addEventListener("이벤트명", 함수정의)
    2. element.addEventListener("click", function(){ 실행할 문장; } )
